❓ 使い方ガイド

1. このアプリでできること

かんたんに在庫や物品の数を記録・カウントし、確認状況を「未確認 / 確認済」で管理できるツールです。

2. 基本の使い方（4ステップ）

  1. 登録する: 画面左側のフォームに「品名」「置き場所」「数量」「単位」を入力して「登録する」ボタンを押します。
  2. 同じ品名を重ねる: すでに登録してある「品名」と「置き場所」が完全に一致するものを再度登録すると、数量が加算（統合）されます。
  3. 数を調整する: カード内の「＋」「ー」ボタンで、すばやく数量を増減できます。
  4. 確認チェック: 数え終わったカードは「確認済」にチェックを入れることで、数え忘れを防止できます。

3. 安全なデータ保存

入力したデータは、インターネットには送信されず、お使いのこの端末（ブラウザ）の内側（localStorage）にのみ保存されます。そのため、通信環境がなくても動作します。

4. 📥 データの保存と書き出し

  • コピー機能: 他のメールや書類に貼り付けられるよう、テキスト形式に変換してクリップボードにコピーします。
  • CSVダウンロード: Excel等で開けるCSV形式のファイルをダウンロードできます。
