🔎 差分ウォッチ Box版 v0.1β / 誰が・いつ・どれくらい / 属人化を絞る

使い方

Box を「誰が・いつ・どのファイルを」更新したかで調べ、更新頻度から属人化しやすい業務を絞り込むための道具です。ファイルの中身は一切見ません（名前・更新日時・更新者だけ）。

ここをクリックして、スキャン結果（.json）を読み込む
1つ選ぶと「今の状態」／2つ選ぶと「2回のスキャンの差分」が出ます
スキャン結果は box_scan.py（Box を巡回して誰が・いつを記録）が作ります。まずは同梱の _sample_snapshot.json（実データのサンプル）で動きを確認できます。▶ サンプルで見てみる
本人・チームの同意のもと、属人化した業務を見つけて引き継ぎに役立てるための道具です。人の評価・監視には使いません。／ ファイルの中身は読みません。
