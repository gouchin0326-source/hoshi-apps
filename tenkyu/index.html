📱 スマホ
・画面の 左半分 を なぞる → その向きへ うごく（上=前／下=後ろ／左右=よこ）
・画面の 右半分 を なぞる → まわりを 見まわす
・右の ▲▼ ボタン → 上・下へ（左右は 同時に つかえます）
💻 パソコン（見まわし方は ⚙で 2つから えらべます）
・クリックで固定（おすすめ）→ カーソルが 中央に固定＝ずれません・マウスで 見まわす・Escで解除・クリックで 正面の星をひらく
・クリックなし→ マウスを 動かすと 見まわす（カーソルは 自由に動きます）
・感度は ⚙「マウス感度」／上下が逆なら ⚙「上下の向きを 反転する」
・W A S D（矢印キー）→ 前後左右、Space／Shift → 上下、ホイール → 前後
共通：🔍さがす→星まで とぶ／🎬めぐる→じどうで 作品を まわる／星にふれる → 作品がひらいて 曲がながれます
